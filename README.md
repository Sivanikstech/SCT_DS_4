# Task 04 – Traffic Accident Data Analysis

## 📌 Internship
SkillCraft Technology – Data Science Internship

## 📊 Dataset
Traffic Accident Dataset

## 🎯 Objective
Analyze traffic accident data to identify patterns related to road conditions, weather, and time of day, and visualize accident hotspots and contributing factors.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn

## 🔍 Steps Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of accident trends
- Identification of key factors affecting accidents

## 📈 Key Insights
- Accident frequency varies with time of day
- Weather conditions influence accident rates
- Certain road conditions contribute to higher accident risk

## 📂 Files Included
- Analysis code (.ipynb / .py)
- Dataset (CSV file)
- Visualization outputs (graphs)

## 💡 Conclusion
This task helped in understanding real-world data analysis and identifying meaningful patterns that can help improve road safety.
